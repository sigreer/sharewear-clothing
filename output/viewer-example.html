Integration Options:

Option 1: Using model-viewer (Recommended)

<script type="module" src="https://ajax.googleapis.com/ajax/libs/model-viewer/3.4.0/model-viewer.min.js"></script>

<model-viewer
    src="yellow-shirt.glb"
    alt="Product description"
    camera-controls
    auto-rotate
    environment-image="neutral"
    shadow-intensity="1"
></model-viewer>

Option 2: Using Three.js

import * as THREE from 'three';
import { GLTFLoader } from 'three/examples/jsm/loaders/GLTFLoader';

const loader = new GLTFLoader();
loader.load('yellow-shirt.glb', (gltf) => {
    scene.add(gltf.scene);
});

Option 3: Shopify Integration

Performance Tips:

  • Use Draco compression (--draco flag) to reduce file size by ~50%
  • Consider lazy loading for pages with multiple models
  • Host GLB files on a CDN for faster loading
  • For AR experiences, enable camera access on mobile
